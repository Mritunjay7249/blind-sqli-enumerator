# Blind SQL Injection Enumerator

A Python-based automation tool that performs Boolean-Based Blind SQL Injection enumeration on intentionally vulnerable labs provided by PortSwigger Web Security Academy.

## Features

- Username Enumeration
- Password Enumeration
- Custom Character Set
- Fast Automation
- Python Requests
- Clean Output
- Beginner Friendly

## Technologies

- Python
- Requests
- Burp Suite
- PortSwigger Labs

## Purpose

This project was developed for educational purposes to understand how Boolean-Based Blind SQL Injection works and how repetitive testing can be automated.

## Disclaimer

This tool is intended ONLY for educational use on authorized environments such as PortSwigger Web Security Academy labs.

Never use it against systems without explicit permission.
